# Source Code Directory

This directory contains the core source code for the MiMi project.

## Structure

- `mimi-commander/` - Main orchestrator written in Rust
- `pandora-memory/` - Memory management engine in C++
- `beatrice-ui/` - User interface and NLP bridge
- `echidna-lab/` - Skills generator and WASM compiler
- `odlaguna-guard/` - Security monitor and watchdog
- `ryzu-runtime/` - Docker container manager
