**Data**:
```rust
pub struct AdaptiveTimeout {
    pub adapter_name: String,
    pub baseline_ms: u32,        // default 30000

    pub p95_margin_percent: f32, // e.g., 120% = p95 * 1.2

```

**Methods**:
- `new(name: String, baseline_ms: u32) -> Self`

- `reset_to_baseline()`
